Step (1) Clone or Download the Repository
   Click the green Code button on GitHub and download the ZIP file.

Step (2) Get Your API Key
   Sign up on OpenWeatherMap and grab your API key.

Step (3) Add Your API Key to the Project
Open script.js in any text editor.
Find this line:
"const apiKey = "YOUR_API_KEY_HERE";"
Replace "YOUR_API_KEY_HERE" with your actual API key and save the file.
